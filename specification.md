# Propozice festivalu

## 1. Identita festivalu
- **Název:** CodeConnect 2026 - IT networking festival
- **Ročník:** 0. ročník 
- **Lokalita:** Praha – Troja (Loděnice Troja)
- **Datum:** 21.8.2026 až 23.8.2026

## 2. Koncept a charakter akce
### 2.1 Cílové publikum
- Primární: Nadšenci do IT, kteří se chtějí zdokonalit v širším spektru oboru
- Sekundární: Zájemci o oblast IT, kteří chtějí získat přehled o aktuálních trendech a možnostech kariérního rozvoje
- Terciární: Rodiny IT specialistů
- Jazyk akce: Čeština

### 2.2 Programové formáty
- Odborné přednášky (AI, cybersecurity, startupy)
- Praktické workshopy (etický hacking, DevOps, cloud)
- Networking zóny
- Panelové diskuze
- Večerní kulturní program (kapela, DJ)

### 2.3 Jednovětá “mise”
Cílem festivalu je bourat stereotypy o IT komunitě a vytvářet prostor pro otevřený networking, sdílení zkušeností a profesní růst

## 3. Struktura webu
Web bude statický (HTML + CSS) a bude obsahově odpovídat API endpointům

### 3.1 Stránky
1. **Úvod (index.html)**
    - krátké představení festivalu, datumy, lokace
    - hlavní CTA (vstupenky / program)
2. **Program (program.html)**
    - zobrazení obsahu po jednotlivých dnech
3. **Účinkující (performers.html + performer-detail.html)**
    - seznam účinkujících a jejich detailní informace
4. **Praktické informace (practical.html)**
    - doprava, ubytování, parkování, areál, bezbariérovost, kontakt
5. **Partneři**
    - seznam partnerů, bez kterých by festival nebyl v takové kvalitě
6. **Kontakt**
    - kontakty na organizátory akce, obecné informcae a technickou podporu při koupi/vrácení vstupenek
